# Ahsan – WordPress Developer Portfolio

A pixel-perfect HTML/CSS/JS clone of the WordPress site, ready for GitHub Pages or Vercel deployment.

## 🚀 Deploy in 2 Minutes

### Option 1: GitHub Pages
1. Create a new repo on GitHub (e.g. `ahsan-portfolio`)
2. Upload all 3 files: `index.html`, `style.css`, `script.js`
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Your site is live at: `https://yourusername.github.io/ahsan-portfolio`

### Option 2: Vercel (Recommended – faster)
1. Go to [vercel.com](https://vercel.com) → Sign up with GitHub
2. Click **New Project → Import Git Repository**
3. Select your repo → Click **Deploy**
4. Done! Live in under 60 seconds with a free `.vercel.app` URL

### Option 3: Drag & Drop (Vercel)
1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag your project folder directly onto the page
3. Instantly deployed!

## 📝 Notes on Images
The site references images from the original WordPress hosting.
If those images are unavailable, fallback content is shown automatically:
- Profile photo → auto-generated avatar
- Tech logos → styled text pills
- Portfolio thumbnails → gradient placeholder

To use your own images, replace the `src` URLs in `index.html` with local files.

## 🎨 Customization
All colors are in `style.css` under `:root` variables:
- `--accent`: Main purple (#7c5cfc)
- `--accent2`: Secondary blue (#5c8aff)
- `--bg`: Background (#09090f)
